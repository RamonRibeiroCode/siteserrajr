Comandos git essenciais

- Adicionar e commitar arquivos na máquina local
	git commit -am "Mensagem do Commit" 

- Commitar arquivos no repositório online
	git push

- Atualizar projeto local com repositorio online
	git pull


Comandos git não essenciais

- Verificar commits
	git log
	
- Verificar arquivos em mudança
	git status
